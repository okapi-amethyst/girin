## Riot Games 規約の制約

[Legal Jibber Jabber](https://www.riotgames.com/en/legal) / [Developer Portal Policy](https://developer.riotgames.com/policies/general) に基づく。

### ✅ 許可されている

| 方法 | 条件 |
|---|---|
| 広告収入（パッシブ） | バナー広告、プレロール広告等。「不適切な広告」はRiotが判断 |
| 寄付・投げ銭 | 個人への寄付の募集はOK |
| API利用プロダクトの課金 | Developer Portal登録＋Approved/Acknowledged。**無料枠必須**。課金部分は「変革的（transformative）」であること |

### ❌ 禁止されている

| 方法 | 理由 |
|---|---|
| ペイウォール（有料限定コンテンツ） | Patreon有料限定記事等は明確にNG |
| クラウドファンディング | プロジェクトの資金集め全般がNG |
| Riot IPを使ったグッズ販売 | LoL/2XKOのキャラを使った商品は不可 |
| NFT/暗号通貨 | 一切禁止 |
| ギャンブル要素 | 不可 |

### GIRINプロジェクトに適用すると

```
✅ Webサイトに広告を置く
✅ Ko-fi / Buy Me a Coffee で寄付を募る
✅ 無料枠がある前提でのプレミアム機能（変革的なもの）
✅ Riot IPを使わないオリジナルグッズ（GIRINロゴ等）
✅ ゲーミングデバイスのアフィリエイト

❌ 「GIRINの第2層以降は有料」のようなペイウォール
❌ 2XKOキャラのグッズ販売
❌ Patreonの有料限定コンテンツ
```

## 収益化手段一覧

### Tier 1：すぐにできる（コスト0〜低）

| 方法 | 仕組み | 月収目安 | 注意点 |
|---|---|---|---|
| **Ko-fi / Buy Me a Coffee** | サイト・Discordに寄付ボタン設置。Ko-fiは手数料0% | 数百〜数千円 | 既存ファンがいないと厳しい |
| **Webサイト広告** | Google AdSense等のバナー広告 | 1000PVあたり100〜300円 | PVが少ないうちはほぼ無収入 |

### Tier 2：コミュニティが育ったら

| 方法 | 仕組み | 月収目安 | 注意点 |
|---|---|---|---|
| **ゲーミングデバイスのアフィリエイト** | アケコン、レバーレス、モニター等のAmazonアフィリエイトリンク | 購入額の2〜5% | Riot IPではないので規約OK |
| **GIRINスポンサー制度** | 対戦会やクリエイターがキャラページのスポンサーに。「このページは○○対戦会の提供です」 | 月500〜1000円/枠 | コミュニティ内の相互支援。広告ではなく文化に馴染む形 |
| **スポンサーシップ** | ゲーミングブランド（レバーレス、モニター等）からの協賛 | 案件次第 | 一定の影響力が必要 |

### Tier 3：規模が大きくなったら

| 方法 | 仕組み | 月収目安 | 注意点 |
|---|---|---|---|
| **プレミアム機能（無料枠あり）** | 詳細統計ダッシュボード、コンボ比較ツール、練習スケジュール提案等 | サブスク次第 | **コンボ情報自体は必ず無料**。課金部分は「変革的」であること |
| **API提供** | コンボデータを他の開発者に提供（有料API） | 利用者次第 | Developer Portal登録が必要な場合あり |
| **オリジナルグッズ** | Riot IPを使わないGIRIN独自ブランドのデザイン | — | 2XKOキャラは使えない。GIRINロゴ等ならOK |

## 「有料にできないなら何を有料にするか」

Riot規約の核心は「コンテンツ自体にペイウォールを置くな」。
逆に言えば「変革的な付加価値ツール」は有料にできる。

```
コンボ情報（必ず無料）        付加価値ツール（有料可）
━━━━━━━━━━━━━━━━━━    ━━━━━━━━━━━━━━━━━━
コンボ動画・入力・ダメージ    自分のクリア履歴の統計分析
逆引き検索                   他プレイヤーとの進捗比較
全キャラ全層のガイド          カスタムコンボプラン作成
                             練習スケジュール提案
                             ↑
                         「新しい情報・洞察・体験」
                         ＝ 変革的（transformative）
```

| 規模 | Web | Discord Bot | DB | 合計/月 |
|---|---|---|---|---|
| 〜1,000人/月 | 無料（Vercel/Cloudflare） | 無料（Oracle Free Tier） | 無料（Supabase Free） | **¥0** |

| 〜100,000人/月 | ¥1,500〜¥5,000 | ¥3,000〜¥10,000 | ¥3,000〜¥15,000 | **¥7,500〜¥30,000** |

## フェーズごとの収益化ロードマップ

> プロジェクトPhaseの正の定義は PROJECT_PROMPT.md を参照。Phase 4 は収益化固有の展望。

```
Phase 0-1（検証〜最小構成）
━━━━━━━━━━━━━━━━━━━
コスト: ¥0（無料枠で運用）
収益: 不要
→ まずコンセプト検証と利用者獲得に集中

Phase 2（成長期）
━━━━━━━━━━━━━━━━━━━
コスト: ¥500〜¥3,500/月
収益: Ko-fi寄付ボタン設置 + Web広告開始
→ 寄付だけで維持できれば十分

Phase 3（拡大期）
━━━━━━━━━━━━━━━━━━━
コスト: ¥7,500〜¥30,000/月
収益: 広告 + アフィリエイト + スポンサー制度
→ 複数の収入源で安定的にインフラを賄う

Phase 4（多言語展開）
━━━━━━━━━━━━━━━━━━━
コスト: ¥15,000〜¥50,000/月
収益: 上記 + プレミアム機能（変革的ツール） + API提供
→ グローバル規模ではプレミアム機能が現実的な選択肢に
```
