# 収益化設計

> 最終更新: 2026-04-11
> 方針: 寄付・広告・アフィリエイトの3手段のみ。プレミアム機能・Tier制・スポンサー制度は行わない。
> 収益化の目的は「インフラ支出を賄うこと」。超過収益は追わない。

---

## Riot Games 規約の制約

[Legal Jibber Jabber](https://www.riotgames.com/en/legal) / [Developer Portal Policy](https://developer.riotgames.com/policies/general) に基づく。

| 可否 | 方法 |
|---|---|
| ✅ | 広告収入（バナー広告等） |
| ✅ | 寄付・投げ銭（Ko-fi 等） |
| ✅ | Riot IP を使わないアフィリエイト（ゲーミングデバイス等） |
| ❌ | ペイウォール（コンボ情報の有料限定） |
| ❌ | クラウドファンディング |
| ❌ | Riot IP を使ったグッズ販売・NFT・ギャンブル |

**コンボ情報は必ず無料。**

---

## 収益化手段（3つのみ）

### 1. 広告（Google AdSense 等）

- サイト内のバナー広告枠
- 収益目安: 100〜300円 / 1,000PV（日本語サイト一般値）
- Phase 2〜。Phase 1 は無収益運用

### 2. 寄付（Ko-fi）

- サイト・Discord に Ko-fi ボタンを設置（手数料 0%）
- 金額は支援者任意。強制しない
- Phase 2〜

### 3. アフィリエイト（ゲーミングデバイス）

- レバーレス・アーケードスティック・モニター等の Amazon アフィリエイトリンク
- Riot IP ではないため規約上問題なし
- 収益目安: 商品単価の 2〜3%（例: 3万円のレバーレス → 900円/件）
- コンテンツ文脈に合った機器紹介ページに配置
- Phase 3〜（コミュニティが育ってから）

## インフラ支出予測

| フェーズ規模の目安 | Web ホスティング | Discord Bot | DB（Supabase 等） | 合計/月 |
|---|---|---|---|---|
| 〜1,000人/月（Phase 1） | 無料 | 無料 | 無料 | **¥0** |
| 〜10,000人/月（Phase 2） | 無料〜¥500 | ¥500〜¥1,500 | 無料〜¥1,500 | **¥500〜¥3,500** |
| 〜100,000人/月（Phase 3） | ¥1,500〜¥5,000 | ¥3,000〜¥10,000 | ¥3,000〜¥15,000 | **¥7,500〜¥30,000** |

※ アーキテクチャ次第でコストは大幅に下げられる（Oracle Cloud 無料枠等）。上限値は安全マージンを含む。

## 収入目安の算出（支出カバーに必要な最低ライン）

### Phase 2（支出目安: ¥500〜¥3,500/月）

| 手段 | 支出 ¥3,500 を賄うのに必要な条件 |
|---|---|
| 広告のみ | 11,700〜35,000 PV/月（RPM 100〜300円で試算） |
| 寄付のみ | 500円/杯として 7件/月 |
| **広告＋寄付の複合** | **PV 5,000〜10,000 + 寄付 2〜3件/月 → 現実的に達成可能** |

→ Phase 2 の収益化目標: **広告＋寄付で月¥3,500 をカバーすること。**
PV 1万・Discord 500人規模なら複合で達成できる見込み。

### Phase 3（支出目安: ¥7,500〜¥30,000/月）

| 手段 | 支出 ¥30,000 を賄うのに必要な条件 |
|---|---|
| 広告のみ | 100,000〜300,000 PV/月 |
| アフィリエイトのみ | 3万円商品が 34件/月（非現実的） |
| **3手段の複合** | **PV 5万 + アフィリエイト月10件 + 寄付月5件 → ¥20,000〜¥30,000 圏内** |

→ Phase 3 の収益化目標: **3手段の複合でインフラ支出をカバーすること。**
単独手段への依存はリスクが高いため、3手段をバランスよく活用する。

## フェーズごとのロードマップ

```
Phase 0-1（無収益運用）
━━━━━━━━━━━━━━━━━━━
支出: ¥0（無料枠で全て賄う）
収益: 不要
→ コンセプト検証とコンテンツ充実に集中

Phase 2（広告＋寄付で維持）
━━━━━━━━━━━━━━━━━━━
支出: ¥500〜¥3,500/月
収益目標: Ko-fi 設置 + 広告開始。月¥3,500 のカバーを目指す
→ 寄付だけで維持できるなら広告は後回しでもよい

Phase 3（3手段フル活用）
━━━━━━━━━━━━━━━━━━━
支出: ¥7,500〜¥30,000/月
収益目標: 広告＋寄付＋アフィリエイトでインフラ維持
→ 支出が収益を下回り続ける限り、追加の収益化手段は不要
```
