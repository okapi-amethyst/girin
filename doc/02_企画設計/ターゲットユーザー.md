## 格ゲーが満たしている動機 / 満たしていない動機

複数のプレイヤー動機モデル（Bartle分類、Quantic Foundry、自己決定理論、Octalysis、
Lazzaroの4つの楽しさ、Amy Jo Kimの行動マトリクス）を横断分析した結果：

### ✅ ゲーム内ランクマッチが既に満たしているもの

| 動機 | 具体例 |
|---|---|
| 競争・支配 | 対戦で勝つ、ランクを上げる |
| 挑戦・成長 | 強い相手に勝てるようになる |
| 興奮 | 逆転勝利、クラッチ |
| 有能感（上級者） | 自分が上手くなった実感 |

### ❌ ゲーム内では構造的に満たされていないもの

| 動機 | 説明 | 対応するモデル |
|---|---|---|
| **つながり・居場所** | 格ゲーのオンライン対戦は孤独。対戦→終了→切断の繰り返し | SDT「関係性」/ Bartle「ソーシャライザー」/ Lazzaro「People Fun」 |
| **発見・探求** | 仕組みを解明する楽しさ。コンボ開発、フレーム調査 | Bartle「エクスプローラー」/ QF「ディスカバリー」/ Lazzaro「Easy Fun」 |
| **達成・コンプリート** | 一覧を埋めていく、全部クリアする、進捗を可視化する | QF「コンプリーション」/ Octalysis「所有感」 |
| **創造・表現** | 自分なりのコンボ、ガイド、動画を作って共有する | QF「デザイン」/ Kim「Express」 |
| **意味・貢献** | 自分の活動が誰かの役に立っている実感 | Octalysis「使命感」/ Lazzaro「Serious Fun」 |
| **自律性** | ランクマ以外の「遊び方」が認められること | SDT「自律性」 |

**→ GIRINが埋めるべきはこの❌の領域**

## ペルソナ再設計

従来の「初心者→上級者」のスキル軸だけでなく、
**何を楽しいと感じるか（動機軸）** を掛け合わせて定義する。

### ペルソナ1：外から来る人（未プレイ・離脱組）

**属性**
- LoLプレイヤー、格ゲー未経験者、または過去に格ゲーで挫折した人
- 「コマンド入力が難しそう」「全部覚えないと対戦できない」と思っている

**満たされていない動機**
- **有能感**（初心者にとって格ゲーは「自分にはできない」の塊）
- **自律性**（「ランクマで勝て」以外の入口がない）

**GIRINでの体験設計**
- SNSで短尺コンボ動画を見る →「え、コマンド入力ないの？」
- F2Pなのでインストールのハードルがない
- 第1層の「これだけコンボ」を1つだけ覚える
- **「✅ これだけ覚えたら対戦行ってOK」で背中を押す**
- 対戦で初めてコンボが決まる → 有能感の獲得

**成功指標**: インストール → 第1層コンボ1つ成功 → 対戦に行く

---

### ペルソナ2：コンボクリアを楽しむ人（達成・コンプリート型）

**属性**
- 2XKOをプレイ中。対戦の勝敗とは別に「コンボを練習してクリアすること自体」が楽しい
- 一覧を埋めていく、チェックを入れていく行為に充実感を感じる
- 必ずしも対戦で勝ちたいわけではない（対戦は楽しむが、トレモも同じくらい楽しい）

**満たされていない動機**
- **達成・コンプリート**（ゲーム内にコンボクリアの進捗を追う仕組みがない）
- **所有感**（「自分はここまでやった」を蓄積・可視化したい）
- **自律性**（「ランク=実力」以外の自分の成長指標がほしい）

**GIRINでの体験設計**
- GIRINのコンボを1つずつクリアしていくこと自体が遊びになる
- **クリア数の可視化**: 自分の進捗をサイト上またはDiscordで確認できる仕組み（具体的な実装方法は実装未定）
- クリア報告の投稿場所（「今日これできた！」を共有する場）
- **他人と比較する順位ではなく、自分の進捗を積み上げる設計**

**成功指標**: GIRINを見る → クリア数を積み上げる → 継続的にGIRINに戻ってくる

**補足：「クリア数をアピールしたい欲」はどこから生まれるか**
- Octalysis「所有感（Core Drive 4）」: 自分が積み上げたものを形として持ちたい
- QF「コンプリーション」: 達成率が上がること自体が報酬
- SDT「有能感」: 「これだけ出来るようになった」の可視化が自己効力感を生む
- これは対戦の勝率やランクとは独立した成長軸になる
- **ポイント: 順位をつけなくても「自分の積み上げ」が見えるだけで十分動機になる**
  - 格ゲーで唯一の成長指標が「ランク」なのが問題。ランク以外の軸を作ることで自律性も同時に満たせる

---

### ペルソナ3：困りごとを解決したい人（実用・課題解決型）

**属性**
- 2XKOをプレイ中。対戦で具体的に困っていることがある
- 「端に追い詰めたのにダメージが安い」「崩した後何すればいい？」
- Wikiを見ても情報が多すぎてどれが実用的かわからない

**満たされていない動機**
- **有能感**（やり方がわからないから無力感がある）
- **自律性**（自分で問題を特定して解決したい）

**GIRINでの体験設計**
- 困りごとから逆引きで状況別コンボにたどり着く
- 第2層で1つ覚えて対戦に戻る
- **「全部覚えなきゃ」ではなく「今困っていることだけ解決すればOK」**
- 解決→対戦→新しい困りごと→GIRINに戻る（学習ループ）

**成功指標**: 困りごと発生 → GIRINで解決 → 対戦に戻る → リピート

---

### ペルソナ4：コンボ職人・コンテンツクリエイター（探求・創造・貢献型）

**属性**
- コンボ開発自体が遊び。新ルートの発見、Fuse別の最適化に没頭する
- 自分の発見を共有したい。動画を作っている、または作りたい
- コミュニティに貢献したい気持ちがある

**満たされていない動機**
- **発見・探求**（新しいコンボルートを見つける知的好奇心）
- **創造・表現**（自分のコンボを形にして世に出したい）
- **意味・貢献**（自分のコンボが誰かの役に立つ実感）
- **つながり**（同じ情熱を持つ人との交流）

**GIRINでの体験設計（報酬システム）**

| 報酬カテゴリ | 具体的な仕組み | 満たす動機 |
|---|---|---|
| **トラフィック還元** | GIRINの動画枠を投稿者のYouTube（時間指定リンク）に直リンク。再生数が投稿者に還元される | 創造・表現 |
| **SNS露出** | GIRIN公開時にXで「このコンボは@○○さん考案」とメンション。フォロワー増の導線 | つながり・表現 |
| **命名権** | 採用コンボに考案者の名前がつく（「○○式エココンボ」）。格ゲー文化の伝統に倣う | 所有感・意味 |
| **Discord称号** | 採用回数に応じたDiscordロール（実装未定） | 達成・コンプリート |
| **実戦採用フィードバック** | 「あなたのコンボで勝てました」報告を集めて投稿者に共有 | 意味・貢献 |
| **次回お題の提案権** | 月間MVPが翌月コンテストのお題を決められる | 自律性 |
| **コラボ配信出演** | コンボ採用者を配信にゲスト招待。考案コンボの解説役として | つながり・表現 |

**最重要の構造的インセンティブ:**
```
投稿者のYouTubeにGIRINがリンクする
→ GIRINの利用者が増えるほど投稿者の再生数も増える
→ 投稿者が自発的にGIRINをSNSで宣伝する動機が生まれる
→ GIRINの認知度が上がる → 利用者が増える → 再生数が増える...
（Win-Winの好循環）
```

**成功指標**: コンテスト参加 → コンボ採用 → YouTube再生数増 → 次回も参加

```
                    GIRINの機能
                        │
    ┌───────────────────┼───────────────────┐
    │                   │                   │
 コンボガイド       クリア進捗          コンテスト

    │                   │                   │
    ├→ 有能感          ├→ 達成感          ├→ 探求・発見
    ├→ 自律性          ├→ 所有感          ├→ 創造・表現
    └→ 課題解決        └→ 自律性          ├→ 意味・貢献
```

### ゲーム内が満たすもの vs GIRINが満たすもの

```
ゲーム内ランクマッチ          GIRIN
━━━━━━━━━━━━━━━━━━━━━    ━━━━━━━━━━━━━━━━━━━━━
✅ 競争・支配                 ✅ つながり・居場所
✅ 挑戦・成長                 ✅ 発見・探求
✅ 興奮                       ✅ 達成・コンプリート
✅ 有能感（上級者）           ✅ 創造・表現
                              ✅ 意味・貢献
                              ✅ 自律性（ランク以外の軸）
                              ✅ 有能感（初心者にも）
```

**→ GIRINは「ランクマの補完」として機能する。競合ではなく共存。**

## ペルソナごとの導線（更新版）

```
ペルソナ1（外から来る人）
  SNSでコンボ動画を見る
  →「簡単そう、F2Pだし」→ インストール
  → 第1層コンボ1つ成功 → 対戦へ
  → ペルソナ2 or 3 に移行

ペルソナ2（コンプリート型）
  GIRINのコンボを1つずつクリア
  → クリア数が積み上がる → ロール獲得
  → 新キャラ追加 → 新しいコンボが増える → また戻ってくる
  → 慣れてきたらペルソナ4に移行も

ペルソナ3（課題解決型）
  対戦で困る → GIRINを逆引き
  → 状況別コンボを1つ覚える → 対戦に戻る
  → 新しい困りごと → またる（学習ループ）

ペルソナ4（クリエイター型）
  コンテストに参加 → コンボ投稿
  → 採用 → YouTube再生数増 + 命名権 + ロール
  → GIRINを自発的に宣伝（Win-Win）
  → 次回も参加
```

## 参考：使用したプレイヤー動機モデル

| モデル | 提唱者 | 主な知見 |
|---|---|---|
| Bartle分類 | Richard Bartle (1996) | Achiever/Explorer/Socializer/Killer。プレイヤーの約80%はSocializer |
| Gamer Motivation Model | Nick Yee / Quantic Foundry | 40万人の調査に基づく12動機。Action/Social/Mastery/Achievement/Immersion/Creativity |
| 自己決定理論（SDT） | Deci & Ryan | 自律性・有能感・関係性の3つが内発的動機の源泉 |
| Octalysis | Yu-kai Chou | 8つのコアドライブ。使命感・所有感・社会的影響が格ゲーで最も未開拓 |
| 4つの楽しさ | Nicole Lazzaro | Hard Fun/Easy Fun/People Fun/Serious Fun。People FunとSerious Funが格ゲーの構造的欠落 |
| Social Action Matrix | Amy Jo Kim | Compete/Collaborate/Explore/Express。プレイヤータイプを「動詞」で捉える |
