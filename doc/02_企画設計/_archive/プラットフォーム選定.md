# プラットフォーム選定

> 作成日: 2026-03-26
> 決定: Webサイト（入口）+ Discord Bot（体験の場）のハイブリッド

---

## 選定の前提

### PS5ブラウザ問題
- PS5には公式Webブラウザがない（PS4にはあったが削除済み）
- 隠しブラウザはSonyがアクセス手段を潰しておりBAN対象のリスク
- **→ PS5プレイヤーもオフライン勢も「手元のスマホ」が実質のアクセス手段**
- これは格ゲー勢の実態と一致（攻略は横にスマホ置いて見る）

## 採用構成：Web ＝ 入口、Discord ＝ 体験の場

### なぜWebだけではダメか
Webアプリで全機能を完結させると：
- 見て終わり → Wikiの二の舞
- Discordに来る理由がなくなる → コミュニティが育たない
- 進捗チェックが「ポチッと押すだけ」→ 孤独な体験

### なぜDiscordだけではダメか
- 情報を探すUXが致命的に悪い（チャット形式）
- SEOがゼロ（検索で見つからない）
- 非Discordユーザーに届かない
- オフ会で「チャンネル探して」は使いにくい

### ハイブリッドの役割分担

| | Webサイト | Discord |
|---|---|---|
| 役割 | メニュー表（情報の入口） | 店内（体験の場） |
| コンボ閲覧 | ◎ 誰でもアカウント不要 | △ チャットで探しにくい |
| 検索流入 | ◎ SEO | ✕ |
| QRコード | ◎ | △ |
| クリア報告 | ✕ 意図的に置かない | ◎ 人に見せる体験 |
| ロール・称号 | ✕ | ◎ |
| コンテスト | ✕ | ◎ |
| デュオ募集 | ✕ | ◎ |
| 最新情報 | 遅れて反映 | ◎ 最速 |

**Webだけで満足する人＝それでOK（コンボ見て対戦に行ってくれれば目的達成）**
**もっと関わりたい人＝Discordに自然に流れる（そこにしかない体験がある）**
