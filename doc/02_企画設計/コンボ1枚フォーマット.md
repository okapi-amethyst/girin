# コンボ1枚フォーマット

> GIRINに掲載する1コンボあたりの情報設計
> データ構造の定義は PROMPT_COMBO_DATA.md を参照
> キャラクターIDは CHARACTER_MASTER.md を参照

## テンプレート

```
┌──────────────────────────────────────┐
│ [動画 / GIF]                         │
│                                      │
│ ポイント：[キャラ名]                 │
│ アシスト：[キャラ名 or なし]         │
│ Fuse：[Fuse名 or No Fuse]           │
│ 位置：[中央 / 端付近 / 端 / ...]     │
│                                      │
│ 入力：[入力表記]                     │
│ ダメージ：[数値]                     │
│ 難易度：★1〜5                       │
│ 実用度：[自動算出スコア]              │
│                                      │
│ タグ：[起き攻め可] [端運び] [入替え]   │
│       [フューリー使用] [HKD] [LS]     │
│       [めくり] [確反用]               │
│                                      │
│ パルス：[使用 / 不使用]              │
│ パルスなし版 → コンボ #[ID]          │
│                                      │
│ 📝 投稿：[投稿者名]                 │
│    考案：[考案者名]                  │
│    補足：[メモ]                      │
│                                      │
│ バージョン：[1.2.0]                  │
└──────────────────────────────────────┘
```

## 各フィールドの設計意図

| フィールド | 目的 | データ元 |
|---|---|---|
| 動画/GIF | テキストでは伝わらない | `video_url` |
| ポイント / アシスト | キャラ構成を明示。CHARACTER_MASTER のIDで管理 | `point_character` / `assist_character` |
| Fuse | コンボの前提条件 | `fuse` |
| 位置 | 画面位置の前提条件 | `position` |
| 入力表記 | コマンド。アシスト技の種別もここに含む | `inputs` |
| ダメージ | 定量的な効果 | `damage` |
| 難易度 | ★1〜5の5段階 | `difficulty` |
| 実用度 | 位置・始動・ゲージ消費・タグ数から自動算出 | `practicality` |
| タグ | コンボ後の状況や特徴を分類。確定値: 起き攻め可 / 端運び / 入替え / フューリー使用 / ハードノックダウン / リミットストライク / めくり / 確反用 | `tags` |
| パルス / 関連コンボ | パルス有無と、有/無バージョンの相互参照 | `pulse` / `related_combo_id` |
| デュオ限定 | デュオ専用コンボかどうか | `duo_only` |
| 対象キャラ | 全キャラ対応か、例外/限定があるか | `target_characters` |
| 投稿者 / 考案者 | クレジット。貢献者のモチベーション | `submitter_name` / `creator_name` |
| 補足メモ | サイド依存等の追加情報 | `notes` |
| バージョン | パッチ対応の明示 | `game_version` |
