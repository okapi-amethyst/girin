## Phase 1 設計

### ページ概要

ブックマーク機能は未実装のため、Phase 1 では Fuse別ページと同様のサンプルコンボ一覧を表示する静的ページとして実装する。
ナビゲーション（ヘッダー・各ページの導線）からのリンクは **Phase 1 では設置しない**。URL を直接入力したユーザーのみアクセスできる状態にとどめる。

### コンポーネント構成（Phase 1）

```
mycombo/index.astro
└── Base.astro
    └── .container.page
        ├── nav.breadcrumb（トップ / My Combos）
        ├── header（Phase 2 告知バナー）
        └── section.combo-area
            ├── div.combo-list
            │   └── ComboCard × n（サンプルコンボ）
            └── nav.pagination
```

### Phase 1 の表示内容

- **コンボ一覧:** 既存のサンプルコンボ（`src/data/sample-combos.ts` / `ahri.json` 等）を使用
- **レイアウト:** Fuse別ページのコンボエリアと同じ構成（ComboCard + ページング）
- **フィルター:** なし（サンプルデータのため不要）
- **告知バナー:** 「このページは開発中です。ブックマーク機能は今後実装予定です。」程度のメッセージ

## Phase 2 設計

### ページ概要

ユーザーがブックマークしたコンボを一覧表示する。データはローカルストレージに保存し、サーバーサイドは不要。

### コンポーネント構成（Phase 2）

```
mycombo/index.astro
└── Base.astro
    └── .container.page
        ├── nav.breadcrumb
        ├── header
        │   ├── h1「My Combos」
        │   └── ツールバー（エクスポート / インポート）
        ├── div.pair-selector（ペア選択UI）
        └── section.combo-area
            ├── div.combo-list
            │   └── ComboCard × n（選択ペアのブックマーク済みコンボ）
            └── div.combo-empty（ブックマークなし時 / ペア未選択時）
```

### Phase 2 の機能

- **ペア選択**: ブックマーク済みコンボが存在するペアを一覧表示し、先にペアを選択する。選択後は同一ペアのコンボのみ表示するため、ComboCard へのペアラベル追加は不要
- ブックマーク済みコンボを `ComboCard` で表示
- エクスポート: ブックマークデータを JSON ファイルでダウンロード
- インポート: JSON ファイルからブックマークを復元
- 削除: カードのブックマークボタンでブックマーク解除
- データ: `localStorage` に `{ comboId, pairSlug, fuseSlug }[]` を保存

## 広告配置（Phase 2〜）

### Phase 1

広告なし（noindex・ナビゲーション非公開のサンプルデータページのため）。

### Phase 2 以降

My Combos はユーザーが自分のデータを管理するパーソナルページ。広告の存在感が大きくなりやすいため、他ページより控えめな配置とする。

#### 候補位置

| # | 位置 | セクション間の区切り |
|---|---|---|
| ① | `nav.pagination` の下（コンボリスト読み切り後） | ページング ↓ **[広告]** ↓ フッター |

| 候補 | 推奨サイズ | 備考 |
|---|---|---|
| ① | `300×250` または Responsive | **唯一の候補**。コンボリストを最後まで確認したユーザーにのみ表示 |

#### 特記事項

- **コンボ0件時（`div.combo-empty` 表示中）は広告を非表示にする**: 「ブックマークがありません」と広告だけのページになると UX が著しく悪化する
- `noindex` ページのため AdSense 自動広告は無効化し、手動スロットのみ使用する
- ツールバー（エクスポート / インポート）付近には広告を置かない
