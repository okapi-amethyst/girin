# ペアトップページ 詳細設計

> 対象URL: `/[char1]/[char2]/`
> 実装ファイル: 専用コンポーネント（`PairComboPage.astro` から分離予定）
> 最終更新: 2026-04-12

---

## ページ概要

特定ペアのハブページ。**Fuse 選択への誘導が主目的。**
コンボの全件一覧は持たず、最新コンボ5件のプレビューのみ表示する。
フィルターバー・ページングなし。

---

## ページ構成

```
PairTopPage.astro（PairComboPage.astro から分離する専用コンポーネント）
├── Base.astro
└── .container.page
    ├── nav.breadcrumb
    ├── header.pair-header
    ├── div.headroom-nav      ← pair-header がスクロールアウトしたら表示
    ├── nav.fuse-nav          ← メインCTA。Fuseタブを押してFuse別ページへ
    └── section.recent-combos
        ├── h2 "最近追加されたコンボ"
        └── ComboCard × 5（新着順・Fuse問わず）
```

### フィルターバー・ページング

**不要。** このページはコンボ探索の場ではなく Fuse 選択の入口のため。

---

## 各セクション仕様

### パンくずナビ（`nav.breadcrumb`）

```
トップ > [キャラA名]
```

※ パンくずの canonical 親はキャラ単体ページ（`/[char1]/`）

### ペアヘッダー（`header.pair-header`）

Fuse 別ページと同仕様。キャラ名・アイコン・ナビリンク（`div.pair-nav-links`）を含む。
詳細は [Fuse別ページ.md](./Fuse別ページ.md) を参照。

### Headroom ナビ（`div.headroom-nav`）

構造・スタイルは [ナビ・ヘッダー・フッター.md](./ナビ・ヘッダー・フッター.md) を参照。

**このページ固有の挙動:**
- トリガー: スクロールダウン時 + **`header.pair-header` が画面外に出たとき** 表示
- 表示内容: `キャラA × キャラB`（`href="#"` でページトップへスクロール）
  - 下までスクロールしたとき Fuse タブナビへ戻る手段として機能する
- フィルターバーがないため動画パネルの `top` オフセットは不要

### Fuse タブナビ（`nav.fuse-nav`）

Fuse 別ページと同仕様だが、**ALL タブを持たない。**
ペアトップ自体が「ALL を経由せず Fuse を選ばせる」ページなので、ALL を選択済みにする表示は行わない。

| タブ | リンク先 |
|---|---|
| Double Down | `/[char1]/[char2]/double-down/` |
| 2X Assist | `/[char1]/[char2]/2x-assist/` |
| Freestyle | `/[char1]/[char2]/freestyle/` |
| Sidekick | `/[char1]/[char2]/sidekick/` |
| Sidekick Alt | `/[char1]/[char2]/sidekick-alt/` |

### 最近追加されたコンボ（`section.recent-combos`）

- 見出し: 「最近追加されたコンボ」
- 件数: **5件固定**（ページングなし）
- 並び順: 追加日時の新しい順
- フィルターなし（Fuse・難易度・始動を問わず表示）
- コンボが5件未満の場合は存在する分だけ表示
- コンボが0件の場合はセクションごと非表示

## コンポーネントの分離について

現状は `PairComboPage.astro` を `fuseSlug: null` で流用しているが、
このページ専用の `PairTopPage.astro`（仮称）を作成して分離する。

分離のタイミング: Fuse 別ページから独立したUIを追加するとき（= このレイアウト実装時）。

## OGP / SEO

### ページタイトル
```
{キャラA} × {キャラB} コンボ | 2XKO GIRIN
```
例: `アーリ × ヤスオ コンボ | 2XKO GIRIN`

### メタディスクリプション
```
2XKO {キャラA}×{キャラB}のFuse別コンボ一覧。Double Down・Freestyle・Sidekickなどのコンボを探そう。
```

### OG画像
- キャラアイコン2つ + ペア名 + GIRINロゴ（動的生成）
- サイズ: 800×418

### canonical
- `/{char1}/{char2}/`
