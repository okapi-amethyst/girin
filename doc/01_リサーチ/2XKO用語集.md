# 2XKO 用語集 / Glossary

2XKO の公式用語および格闘ゲーム一般用語をまとめたもの。

---

## ボタン表記（Button Notation）

| 表記 | English | 意味 |
|---|---|---|
| L | Light | 弱攻撃 |
| M | Medium | 中攻撃 |
| H | Heavy | 強攻撃 |
| S1 | Special 1 | 必殺技1 |
| S2 | Special 2 | 必殺技2 |
| T | Team | タッグボタン。パートナー関連の操作に使用 |

---

## タッグシステム（Tag System）

| 日本語              | English              | 意味                                     |
| ---------------- | -------------------- | -------------------------------------- |
| ポイント             | Point                | 現在操作中のキャラ                              |
| アシスト             | Assist               | 控えにいるパートナーキャラ                          |
| ハンドシェイクタッグ       | Handshake Tag        | Tボタンで操作キャラを交代。両キャラが一時的に同時に場に出る         |
| クイックタッグ          | Quick Tag            | T長押しで攻撃なしの即時交代                         |
| アシストアクション        | Assist Action        | 前T / 後Tで控えキャラを呼び出して攻撃させる（各キャラ2種）       |
| チャージアシスト         | Charge Assist        | アシストアクション中にT長押しで強化版を出す                 |
| タッグランチャー         | Tag Launcher         | 下T。相手を打ち上げつつパートナーに交代。ダメージ補正をリセットする     |
| プッシュアシスト         | Push Assist          | ガード中にT。控えキャラが相手を押し返す。メーター不要だがアシストCDを共有 |
| スーパーアシスト         | Super Assist         | パートナーKO後にアシストアクション入力。メーター1本消費で強力な攻撃    |
| ブレイク / ダイナミックセーブ | Break / Dynamic Save | S1/S2+T。コンボブレイカー。相手のコンボ中にアシストを呼んで脱出    |
| フューリーブレイク        | Fury Break           | パートナーKO後に使える強化版ブレイク。                   |
| タッグキャンセル         | Tag Cancel           | 行動をハンドシェイクタッグでキャンセルする                  |


---

## フューズ（Fuse）

| 日本語     | English     | 意味                                                         |
| ------- | ----------- | ---------------------------------------------------------- |
| フューズ    | Fuse        | キャラ選択時に決めるチーム修飾システム。タッグの挙動を変える                             |
| ダブルダウン  | Double Down | 自分のスーパー中に後退やアシストのスーパーでキャンセルできる。初心者向け                       |
| 2Xアシスト  | 2X Assist   | 1回の呼び出しでアシストアクションを2連続で出せる                                  |
| フリースタイル | Freestyle   | ハンドシェイクタッグを1回のシーケンスで2回使える。                                 |
| ジャガーノート | Juggernaut  | ソロモード。チーム機能をブレイク以外すべて無効化。                                  |
| サイドキック  | Sidekick    | ソロ+アシストモード。アシストアクション・プッシュアシスト・スーパーアシスト・ブレイクは使用可。タッグ交代系は不可。 |

---

## フューリー（Fury）

| 日本語            | English          | 意味                                           |
| -------------- | ---------------- | -------------------------------------------- |
| フューリー          | Fury             | 20秒間の強化バフ状態。キャラが赤く光る                         |
| フューリーダッシュキャンセル | Fury Dash Cancel | フューリーメーターを消費して技の硬直をダッシュキャンセル。投げや起き上がり攻撃にも使用可 |

## ゲージ（Gauges）

| 日本語           | English                          | 意味                                                              |
| ------------- | -------------------------------- | --------------------------------------------------------------- |
| スーパーメーター      | Super Meter                      | スーパー(1本)、アルティメット(3本)、パリィ(1本/成功時返却)、スーパーアシスト(1本)に使用。通常上限3本、ソロ時5本 |
| ブレイクメーター      | Break Meter                      | 時間経過と被ダメージで蓄積。満タンでブレイク（コンボブレイカー）使用可能                            |
| 回復可能体力 / 灰色体力 | Recoverable Health (Grey Health) | ダメージの一部が灰色で残り、タッグアウトやフューリー中に回復する                                |

## 攻撃・行動（Offense / Actions）

| 日本語           | English                | 意味                                 |
| ------------- | ---------------------- | ---------------------------------- |
| 通常技 / ノーマル    | Normal                 | L / M / Hボタンで出る基本攻撃                |
| 必殺技 / スペシャル   | Special Move           | 方向+S1/S2で出る固有技。                    |
| スーパー (Lv1)    | Super                  | S1 or S2。メーター1本消費の超必殺技             |
| アルティメット (Lv3) | Ultimate (Lv3)         | S1+S2。メーター3本消費の最強技                 |
| 投げ            | Throw                  | M+H。近距離で相手を掴む攻撃。ガードを崩す手段           |
| クリティカルストライク   | Critical Strike        | H長押し。溜め攻撃でガードブレイク性能を持つ             |
| コマンド投げ / コマ投げ | Command Grab           | コマンド入力で出す投げ技。通常ガード不能               |
| 中段            | Overhead               | 立ちガードでしか防げない攻撃                     |
| 下段            | Low                    | しゃがみガードでしか防げない攻撃                   |
| 飛び道具 / 弾      | Projectile             | 離れた距離を移動する攻撃                       |
| 対空 (2H)       | Anti-Air (AA)          | 下+H。空中攻撃に対してフレーム4から無敵を持つ汎用対空技      |
| めくり           | Cross-up               | 空中で下+Hを使用。ジャンプで相手を飛び越えガード方向を逆にする攻撃 |
| 崩し            | Mix-up                 | 相手のガードを揺さぶる攻撃の読み合い                 |
| 表裏            | Left-Right Mix         | 攻撃が表から来るか裏から来るかの択                  |
| 上下択           | High-Low Mix           | 中段と下段を使い分けてガードを崩す択                 |
| 固め            | Pressure / Blockstring | 相手をガードさせ続ける連続的な攻撃                  |
| 暴れ            | Mashing / Reversal     | ガード中・被攻撃中に強引に攻撃を出すこと               |
| 暴れ潰し          | Frame Trap             | 相手の暴れを誘い、その隙に攻撃を当てる連携              |
| 重ね            | Meaty                  | 相手の起き上がりに攻撃判定を重ねること                |
| 起き攻め          | Okizeme (Oki)          | ダウンした相手の起き上がりを攻める状況                |
| カウンターヒット      | Counter Hit            | 相手の技の発生中や硬直中にヒットさせること。追加効果あり       |
| アーマー          | Armor                  | 一部の技に付与。ダメージ軽減しつつヒットスタンを無視して行動を続行  |
| 削り            | Chip Damage            | ガード時に受ける削りダメージ。フューリー中は通常技にも10%付与   |

## 防御・回避（Defense）

| 日本語      | English          | 意味                                  |
| -------- | ---------------- | ----------------------------------- |

| 後退ガード    | Retreating Guard | ガード中に後ろ入力。自キャラが後退するが下段に弱い           |
| ブレイク     | Break            | S1/S2+T。コンボブレイカー。スーパー/アルティメット中は使用不可 |
| 投げ抜け     | Throw Tech       | 相手の投げを入力で抜けること                      |
| 割り込み     | Interrupt        | 相手の連携の隙間に攻撃を差し込むこと                  |

## コンボ（Combos）

| --------------- | -------------------- | -------------------------------------------------------- |
| コンボ             | Combo                | 相手が抜けられない連続攻撃                                            |
| パルスコンボ / オートコンボ | Pulse Combo          | キャラ選択時にON/OFF可能な簡易コンボ。同じボタン連打で発動                         |
| エリアル / 空中コンボ    | Aerial / Air Combo   | 相手を浮かせて空中で繋げるコンボ                                         |
| 地上コンボ           | Ground Combo         | 地上で繋げるコンボ                                                |
| コンボルート          | Combo Route          | コンボの繋ぎ方・経路                                               |
| コンボレシピ          | Combo Recipe         | 具体的なコンボの手順                                               |
| 始動技             | Starter              | コンボの起点になる技                                               |
| リンク / 繋ぎ        | Link                 | フレーム有利を利用して次の技を繋げること                                     |
| キャンセル           | Cancel               | 技の動作中に別の技で上書きすること                                        |
| ヒットスタン          | Hit Stun             | 被弾後の行動不能状態。同等以上の強度の技で繋がる                                 |
| ヒットリアクション       | Hit Reaction         | ヒットスタンの中でも特殊なアニメーションを伴う長い硬直                              |

| リミットストライク       | Limit Strike         | 1コンボ中に2回目のメジャーヒットリアクションを使うと発動。コンボ強制終了だがボーナスダメージ+メーター25獲得 |

| タンブル            | Tumble               | 空中で回転しながら吹き飛ぶ（メジャーヒットリアクション）                             |
| ダメージスケーリング / 補正 | Damage Scaling       | コンボが長くなるほどダメージが減少する仕組み。タッグランチャーで100%にリセット                |
| 補正切り            | Reset                | コンボ補正をリセットするためにあえてコンボを中断し崩しに移行すること                       |
| ヒット確認           | Hit Confirm          | 攻撃がヒットかガードかを見てから行動を変えるテクニック                              |
| ワンタッチ / TOD     | Touch of Death (TOD) | 一度触れたら相手を倒し切るコンボ                                         |

## フレームデータ（Frame Data）

| 日本語 | English | 意味 |
|---|---|---|
| フレーム (F) | Frame (F) | ゲーム内の最小時間単位（1/60秒） |
| 発生 | Startup | 技の入力から攻撃判定が出るまでのフレーム数 |
| 持続 | Active Frames | 攻撃判定が出ているフレーム数 |
| 硬直 / 全体 | Recovery / Total Frames | 技の動作が終わるまでのフレーム数 |
| ガード時 | On Block | 技をガードさせた後の有利/不利フレーム |
| ヒット時 | On Hit | 技をヒットさせた後の有利/不利フレーム |
| 有利 (+) | Plus / Frame Advantage | 相手より先に動ける状態 |
| 不利 (−) | Minus / Frame Disadvantage | 相手より遅く動く状態 |
| 確定反撃 / 確反 | Punish | 相手の隙が大きい技にガード後確定でヒットする反撃 |

| ゲージ管理     | Meter Management  | メーターの消費と温存の判断          |
| 体力有利      | Life Lead         | 体力で相手より優位な状態           |
| 画面端 / 端   | Corner            | ステージ端。逃げ場がなく攻められやすい    |
| 端運び       | Corner Carry      | コンボで相手を画面端まで運ぶこと       |

## キャラクタータイプ（Character Archetypes）

2XKO公式の7アーキタイプ。

| 日本語      | English     | 意味                             |
| -------- | ----------- | ------------------------------ |
| オールラウンダー | All Rounder | 攻防バランスの取れた万能型（ダリウス、ヤスオ）        |
| ラッシュダウン  | Rushdown    | 近距離での攻めが得意な攻撃型（アーリ、ヴァイ、ワーウィック） |
| ゾーナー     | Zoner       | 遠距離で戦うのが得意（ケイトリン、ジンクス、セナ）      |
| グラップラー   | Grappler    | 投げを主軸とするキャラ（ブリッツクランク）          |
| トラップ     | Trap        | 設置技やギミックで戦うテクニカルなキャラ（ティーモ）     |
| ブルーザー    | Bruiser     | 体力と火力に優れたパワー型（ブラウム、イラオイ）       |
| ミックスアップ  | Mixup       | 多彩な崩しで相手を翻弄するキャラ（エコー、アカリ）      |

---

> **注**: 2XKOは継続的にアップデートされるため、システムの名称や仕様は変更される可能性があります。
