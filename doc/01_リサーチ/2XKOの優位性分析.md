# 2XKOの優位性分析

> 作成日: 2026-03-26
> 調査時点の情報に基づく

## 対格ゲー（SF6, 鉄拳8, GGSTなど）

| 優位ポイント | 詳細 |
|---|---|
| **デュオプレイ** | 2人で1チームを操作。友達と「一緒に戦う」体験は格ゲー史上初 |
| **Fuseシステム** | 5種のFuse選択でキャラペアの戦略が根本的に変わる（Double Down / 2X Assist / Freestyle / Juggernaut / Sidekick） |
| **完全F2P** | SF6（約6,000円+DLC）、鉄拳8（約8,000円+DLC）に対し全キャラ無料解放可能 |
| **簡易入力** | コマンド入力なし（波動拳等不要）。方向＋ボタンのスマブラ寄り操作 |
| **最高峰ネットコード** | GGPO開発者Tom Cannon本人が設計。クライアントサーバー＋ロールバック＋Fair Playモード |

## 対他ジャンル（MOBA, FPS, パーティゲーなど）

| 優位ポイント | 詳細 |
|---|---|
| **2人協力×対戦** | 1画面で2人が息を合わせる密度の高い協力感（FPSデュオとは質が違う） |
| **試合時間が短い** | 1試合数分。LoLの30〜40分、バトロワの20分と比べ圧倒的に手軽 |
| **LoL IPの親しみ** | LoLプレイヤー1億人超。キャラを知っている潜在層が巨大 |
| **クロスプラットフォーム** | PC/PS5/Xbox全対応。Riotアカウントで進捗共有 |

## コンボ観点での優位性（本プロジェクトの核）

| 要素 | 他の格ゲーとの違い |
|---|---|
| **Fuse × タッグの掛け算** | 同じキャラペアでもFuse5種でコンボが全く異なる。SF6や鉄拳では不可能な多様性 |
| **アシストアクション** | オフスクリーンからのアシスト攻撃をコンボに組み込む独自要素 |
| **ハンドシェイクタグ** | コンボ中にキャラ交代して継続。2人協力なら連携の醍醐味 |
| **ダイナミックセーブ** | コンボブレイカーとしてアシストを呼ぶ防御的タグ |
| **入力の簡易さ** | コマンド入力なしのためコンボ自体の参入障壁が低い |

## 現状の課題

- ロースターが10〜11キャラと少ない
- 「2XKO」の名前の認知度が低い（LoL層にも届いていない）
- クロスリージョンプレイ非対応
- 開発チーム半減（2026年2月レイオフ）
- 2026年は5シーズン→3シーズンに縮小
