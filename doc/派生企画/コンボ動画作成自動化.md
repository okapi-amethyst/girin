# コンボ動画作成自動化

> [!note] 位置づけ
> 本筋（GIRIN サービス開発）とは独立した**派生企画**。
> コンボ動画素材を効率よく量産するための個人用ツール整備。

---

## 概要

コンボ動画を1本ずつ手動でトリミングするのが手間なため、**ffmpeg を使って自動化するPythonスクリプト**を作成した。

- **スタート点**：動画後方から見て最後の `silence_end` を拾う
- **エンド点**：動画末尾そのまま
- **確認フロー**：1本切るごとに出力動画を自動再生し、人間が目視チェックしてから次へ進む

---

## 作成したツール

| ファイル | 場所 |
|---|---|
| `combo_trim.py` | `doc/tools/コンボ切り出し自動化/combo_trim.py` |
| `run_combo_trim_watch.bat` | `doc/tools/コンボ切り出し自動化/run_combo_trim_watch.bat` |

### フォルダ構成

```text
動画編集フォルダ/
└─ コンボ切り出し自動化/
   ├─ combo_trim.py
   ├─ run_combo_trim_watch.bat
   └─ watch/
```

このプロジェクト内では `doc/tools/コンボ切り出し自動化/` にツール本体だけを置いておき、実際に使うときはフォルダごと任意の作業場所へコピーする。

### 運用手順

1. `doc/tools/コンボ切り出し自動化/` を、任意の `動画編集フォルダ/コンボ切り出し自動化/` にコピーする
2. コピー先の `watch/` に、切りたい動画を複数入れる
3. `run_combo_trim_watch.bat` をダブルクリックする
4. 1本目の動画を自動カットして `watch/output/` に出力
5. 出力動画が自動再生される
6. 人間が内容を確認し、プレイヤーを閉じる
7. コンソールで Enter を押すと次の動画の処理が始まる

### 処理後の振る舞い

- 成功した元動画は `watch/done/` に移動
- エラーになった元動画は `watch/error/` に移動
- 出力動画は `watch/output/` に `{元ファイル名}_trimmed.xxx` で保存

## 参考：よくあるズレのパターン

| 症状 | 試すこと |
|---|---|
| スタートが早すぎる | `START_OFFSET` のマイナス幅を小さくする |
| スタートが遅すぎる | `START_OFFSET` をよりマイナスにする |
| 途中の無音を拾って切り始める | `SILENCE_DURATION` を長くする / `SILENCE_NOISE` を下げる |
| 末尾近くの無音を誤検出する | `IGNORE_TAIL_SEC` を増やす |
| そもそも無音が拾えない | `SILENCE_NOISE` を上げる or 録画時の音量バランスを見直す |
