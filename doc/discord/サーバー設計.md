# Discordサーバー設計

> コンボGIRINの総本山となるDiscordサーバー。Botとは独立した設計。

## サーバーの役割

- コンボGIRINのコミュニティハブ
- Web・Botからの誘導先
- コンボの修正・削除依頼の受付窓口
- 投稿者（コンボ職人）の活動拠点

## チャンネル構成

### 情報系（読み専）

| チャンネル | 用途 |
|---|---|
| `#お知らせ` | サイト更新・新機能・パッチ対応情報 |
| `#ルール` | サーバールール・利用規約 |
| `#自己紹介` | 任意。使用キャラ・ランク帯など |

### コンボ関連

| チャンネル | 用途 |
|---|---|
| `#コンボ修正依頼` | Webサイトからの誘導先。削除・修正・バージョン切れ報告 |
| `#コンボ相談` | コンボ提案・「このキャラでこの状況どうする？」等の質問。正式投稿は Web フォーム（Phase 2） |

### コミュニティ

| チャンネル | 形式 | 用途 |
|---|---|---|
| `#雑談` | テキスト | 自由会話 |
| `#相方募集` | フォーラム | デュオ相方探し。キャラタグで絞り込み。Webサイトのコンボカードから誘導 |
| `#対戦募集` | テキスト | カジュアル対戦・練習相手募集 |
| `#クリップ共有` | テキスト | プレイ動画・スクリーンショットの共有 |

### 運営（管理者のみ）

| チャンネル | 用途 |
|---|---|
| `#運営` | 管理者用。サイト運営の作業ログ・判断メモ |

## ロール設計

最小限でスタート。必要に応じて追加。

| ロール | 付与条件 | 権限 |
|---|---|---|
| 管理者 | サーバーオーナー | 全権限 |
| コンボ投稿者 | コンボが1つ以上採用された人 | 発言が目立つ色付き |
| メンバー | サーバー参加者 | 通常権限 |

## Botとの関係

- **GIRIN Bot**（配布型）: `/combo`, `/mycombo` — サーバー内でも使える
- **サーバー運営Bot**（将来・別Bot）: 大会管理・募集管理・ロール自動付与など — 本サーバー専用

Phase 1ではサーバー運営Botは不要。手動運営で開始。

## 導線（Webサイトから）

```
Webサイト
  ├── マイコンボページ下部「このキャラでデュオ相方を探す」→ #相方募集
  ├── 「コンボの修正・削除依頼はこちら」→ #コンボ修正依頼
  ├── 「コミュニティに参加」→ サーバー招待リンク
  └── 「コンボを投稿したい」→ Webフォーム（`/submit/`・Phase 2）
```

## 運営方針

- 小規模スタート。管理者1人で回せる規模を維持
- チャンネルは必要になったら増やす（最初から多くしない）
- 荒らし対策: 認証レベル（メール認証済み）を設定
